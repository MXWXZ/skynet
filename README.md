# skynet

## Develop
### Create plugin
1. Create folder under `plugins`, e.g. `plugins/myplugin`
2. Copy `template.tpl` to `plugins/myplugin/main.go`
3. Read instructions in file to create your plugin
4. Run `go build -buildmode=plugin .` to build your plugin
